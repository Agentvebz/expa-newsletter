04
🎓 Skill of the week: stop saving prompts in Docs
If you've been pasting your "best prompt for X" into a Google Doc and copying it back every time — there's a much better way. Claude Skills are reusable instruction sets you save once and trigger with a slash command in any chat.
Setup: Customize → + → Create Skill Three parts: 1. Description + when Claude should fire it 2. Instructions 3. Two or three example outputs Activate: type /[skill-name] anywhere
Rule of thumb: build a Skill only when the task is recurring, structured, and has a consistent output format. Otherwise a saved prompt is fine.
